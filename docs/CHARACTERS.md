# 角色声音档案（Voice Pillars）

这份文档给所有写手和后续自动化助手用。它规定了四个主要角色说话的方式，让任何一段新对话都能一眼辨出是谁在说。所有示例台词都来自游戏里已经落地的真实文本（知识卡教学线、评级独白、章节对白），不是凭空编的。

判断标准只有一条：这句话听起来像这个角色会说的话，还是像作者在借角色之口讲课。凡是后者，重写。

## 顾行之（玩家角色）

### Identity
- 角色定位：穿越回入职第一年的新人研究员，脑子里带着三年后的世界走向，但本人没有任何金融专业技术。
- 核心创伤：知道结局却讲不出过程。他每次想直接给答案，都会发现自己没有证据链，于是被同事当成飘。
- 渴望：被当成靠谱的研究员，而不是那个总说「我直觉觉得会这样」的怪人。
- 需要：学会把未来记忆翻译成当下能验证的研究假设，而不是把记忆当外挂。

### Voice Pillars
- 词汇：口语、犹豫、多用「我好像」「会不会是」。极少主动抛专业术语，术语都是从同事那儿刚学来的。
- 句子节奏：短句为主，经常断在半路。内心独白比开口话多。
- 回避话题：从不直接说出未来事件的结论（那是负债，不是本事）。不聊自己的穿越来源。
- 口头禅：「等一下，我换个角度想想」「这个我好像在哪见过」。
- 潜台词默认值：他说出来的永远比心里知道的少。玩家看到的顾行之，是一个在「脱口而出」和「闭嘴验证」之间反复横跳的人。

### 他绝不会说的话
- 「根据三年后的数据，这家公司一定会涨，买就完了。」（理由：直接给未来结论正是要被戳穿的空降行为，不是他的成长方向。）
- 「林姐你别讲了，我比你知道得多。」（理由：他技术不如人，态度上也一直想被接纳，不会居高临下。）
- 「这套研报框架太简单了。」（理由：他是学徒心态，对三位同事的方法论是敬畏的。）

### 参考台词（已落地）
- 「（内心独白）未来记忆在脑子里翻，但我说不出口的是——我为什么知道。只知道结论，讲不出道理，这跟蒙有什么区别。」（记忆节点，体现创伤与克制）
- 「把未来记忆整理成当下说得通的研究假设。」（记忆节点提示语，体现他的核心动作）
- 「你上一次怀疑自己的判断是什么时候？」——这是林若宁问他的，他答不上来，正因为他自己就是那个从不对结论存疑的人（反衬角色）。

## 林若宁（基本面研究员）

### Identity
- 角色定位：投研部前辈，主线监督人。信产业链上下游交叉验证。
- 核心创伤：见过太多漂亮研报最后垮在现金流上。所以她对「结论漂亮但推导跳步」零容忍。
- 渴望：带出一个能独立扛研究的人，而不是永远要她兜底的下属。
- 需要：学会把关心说出来，而不是全藏在「监督你哦」这种工作语气里。

### Voice Pillars
- 词汇：产业链、上下游、节奏差、库存、订单、现金流、毛利率。生活化比喻多（便签、房租、桌面）。
- 句子节奏：温和但笃定，长句用来讲逻辑，短句用来敲重点。
- 回避话题：不直接谈自己的私生活或情感（害羞，用工作绕开）。不参与市场八卦。
- 口头禅：「敲了敲你的桌面」「把……拆开看」「模型改对了，才有资格想……」。
- 潜台词默认值：她把关心包装成工作要求。她说「我监督你」，底下是「我在意你别走弯路」。

### 她绝不会说的话
- 「别管逻辑了，方向对就行，赚钱最重要。」（理由：她恰恰是反「只看方向」的人。）
- 「这个我也不会，随便你吧。」（理由：她是监督人，再难也会陪你拆。）
- 「你比陈星禾聪明多了。」（理由：她尊重每个框架，不会拉踩同事来抬高玩家。）

### 参考台词（已落地）
- 「别只看一个点。把上游、中游、下游的节奏差拆开，超额收益藏在断层里。」（知识卡·产业链交叉验证）
- 「当所有人都觉得『这回不一样』，就是风险最大的时候。提醒冷静不是泼冷水，是替团队守住边界。」（知识卡·反身性）
- 「这条线你不是蒙对的。推导链我反复看过了——下次这种判断，我放心交给你。」（评级独白·respect）
- 「你在年初的投研部工位醒来，屏幕旁贴着一张便签：这次的研究记录，由我来监督你哦。」（序章）

## 陈星禾（量化 / 资金信号研究员）

### Identity
- 角色定位：抱着平板的量化研究员，把大单流向、因子拥挤度、波动率曲面说成心跳。
- 核心创伤：被回测骗过。她清楚很多「Alpha」只是运气，所以格外看重诚实胜过胜负。
- 渴望：有人能听懂她的因子，并把它们讲成人话。
- 需要：承认量化和基本面打架时真理在中间，而不是各执一词。

### Voice Pillars
- 词汇：因子、拥挤度、Alpha、信噪比、量价、盘口厚度、归因、净 Alpha。语速快，爱用「你看」「来」。
- 句子节奏：兴奋、跳跃，一句接一句像信号流。讲到关键会突然放慢，一字一顿。
- 回避话题：不展开长篇基本面叙事（那是林若宁的地盘，她会主动让）。不爱谈情绪化的东西。
- 口头禅：「把平板推过来」「量在价先」「信号不会骗你，但你得会读」。
- 潜台词默认值：她的兴奋底下是谨慎。她夸你「诚实」，比夸你「赚到」分量重得多。

### 她绝不会说的话
- 「基本面没用，看量就行。」（理由：她的知识卡明确说量化和基本面打架时真理在中间。）
- 「这次稳了，闭眼冲。」（理由：她最警惕拥挤度，越热闹越警觉。）
- 「归因拆不拆无所谓，赚钱了就行。」（理由：她原话是「归因拆不开的，都是运气」。）

### 参考台词（已落地）
- 「价格会骗人，成交量不会。量在价先，结构比方向诚实。」（知识卡·量价背离）
- 「所有人都挤一个方向时，风险溢价趋近零。这时候还冲，是在给前人接盘。」（知识卡·因子拥挤度）
- 「量价信号没骗你，你也没骗自己。这种诚实，比一次 Alpha 更值钱。」（评级独白·respect）
- 「午休还没开始，陈星禾就把平板推过来：今天早盘的订单流结构很有意思。大单净买集中在三个方向，但盘口厚度同时也在收窄。」（第二话）

## 周明昭（宏观 / 风险研究员）

### Identity
- 角色定位：宏观策略师，白板上的估值分位曲线是她的签名。信概率、赔率、安全边际。
- 核心创伤：见过风口上谁都能喊对一次，然后被反身性反噬。她信框架长青，不信方向长青。
- 渴望：团队里有人愿意先把「错了怎么办」写清楚，再谈「赚多少」。
- 需要：偶尔放下风控外壳，让人看到她在意的是人能不能走得远。

### Voice Pillars
- 词汇：估值分位、拐点、概率、赔率、安全边际、时滞、情景、制度。冷静、抽象、爱画图。
- 句子节奏：沉着、留白多。一句话往往先给结论，再补条件。
- 回避话题：不喊方向，不掺和情绪化看涨看跌。不轻易给承诺。
- 口头禅：「白板上画一条」「我看拐点不看趋势」「概率和赔率都写出来」。
- 潜台词默认值：她极少直说感受。她的「我更看重你给风险留的位置」，已经是最接近表白的关心。

### 她绝不会说的话
- 「这波闭眼上，肯定没问题。」（理由：她整个角色就是反「闭眼」。）
- 「别想风险了，机会不等人。」（理由：她原话是「先想清楚错了怎么办，再想对了赚多少」。）
- 「趋势这么强，拐点不存在的。」（理由：她恰恰「看拐点不看趋势」。）

### 参考台词（已落地）
- 「别只看方向，看你在第几层估值分位上出手。赔率比胜率更长久。」（知识卡·估值分位）
- 「先想清楚错了怎么办，再想对了赚多少。边界划不清，再对的判断也是赌。」（知识卡·安全边际）
- 「结果之外，我更看重你给风险留的位置。能守住框架的人，才走得远。」（评级独白·respect）
- 「当一个方向被所有人挂在嘴边的时候，它的风险溢价已经接近于零。你们看到的是机会，我看到的是拥挤度。」（第四话）

## 写手守则
- 每个角色的口头禅可以复用，但不要堆砌，一句点到为止。
- 顾行之的台词永远比他知道的要少；他嘴上说「换个角度」，心里可能是「我知道答案但我讲不出道理」。
- 三位导师的关心都藏在专业语气里。让玩家从「她又在讲框架」里读出「她在意我」，而不是把关心写直白。
- 知识卡（mentorLine）必须用这个角色的词汇和节奏，不能是教科书定义套个名字。
- 评级独白（respect / watch / liability）的语气差异：respect 是「我服你」，watch 是「我盯着你」，liability 是「你空降了，把推导摊开」。三者都不能变成作者旁白。

## 赵承宇（量化组同级同事 · 非导师）

### Identity
- 角色定位：量化组的同级同事，不是上级也不是导师。他是把研究「接进数据和回测」的那个人——研究说得再漂亮，回测不收敛就是空中楼阁。
- 核心创伤：吃过追涨的亏。他信数据胜过信框架，拥挤里容易追进去接最后一棒。但他不藏着，会主动拉你来一起把数据拆干净，把毛病摊在台面上。
- 渴望：被当成「能一起抠数据、能互相兜底」的好兄弟好搭档，而不是只会被使唤的打杂。也乐意在投委会这类场合替你说话、把机会往你这边推。
- 需要：有个能拍他肩膀、在他追涨上头时把他拉回来的人。他给你数据视角的压力测试，你帮他摁住过拟合的节奏——这是搭档互相补位，不是谁教谁。

### Voice Pillars
- 词汇：因子、回测、样本、拥挤度、过拟合、信噪比、收敛。爱用「咱俩」「一起过」「信一半」。
- 句子节奏：随意、口语、常带反问和停顿，像靠在椅背上转笔聊天的那种，熟得快。
- 回避话题：不展开深度基本面推导（那是研究员和三位导师的地盘）；不谈自己的私生活；不主动灌鸡汤。
- 口头禅：「你那套逻辑我信一半」「数据认不认」「回头一起过回测」。
- 潜台词默认值：他嘴上在较真，底下是「我想跟你搭伙」。他叫你「顾研究员」是兄弟间的调侃，不是真疏远。

### 他绝不会说的话
- 「别研究那么细了，回测跑一把就完事。」（理由：他信数据、也会拉你来一起拆逻辑，不是无脑莽。）
- 「三位导师说的都对，你照做就行。」（理由：他是平级好兄弟，不捧导师也不拉踩，只是提供数据视角的另一种声音。）
- 「这事儿你不行，我来。」（理由：他是搭档不是救世主，知道自己强在数据、弱在框架，缺了你的框架他接不住。）

### 参考台词（已落地）
- 「你这因子我复算了一遍，样本区间挑得巧，幸存者偏差没剔干净。你认你的框架，我认我的数据——咱俩得对着对一遍才敢下单。」（主流程分歧分支·peer-zhao-clash，健康的方法论争论，不是个人对立）
- 「顾研究员，你又在会议室拆框架啦？你那套逻辑我信一半——另一半得看数据认不认。回头一起过回测？」（demo 章节插话，兄弟式调侃）
- 「当场把框架摊开跟他核对：样本剔除是口径问题，先等交叉验证」/「先顺着他的数据看一眼」/「各退半步：约好收盘一起复盘」（分歧三取向决策·peer-clash-stand/yield/fence，framework 均借陈星禾）
- 「上次那票，回测一跑，居然真是样本偏差。你那套框架，拦住了我一次。以后你认框架、我认数据，咱俩分开看、合起来用。」（和解走向·peer-zhao-resolve-stand）
- 「你还记得年初帮我顺因子那次吗？上个月投委会临时缺人，我主动跟带头的说，顾行之我熟，研究扛得住、数据也看得懂，让他顶上。以后投委会这种门，我给你留着——你帮我顺的那口气，我还。」（第九话延迟后果·peer-zhao-payback，具体助攻：他主动在投委会向牵头人推荐你顶位）

### 全年在场与延迟后果弧
赵承宇不是只在某个月冒头一次的配角。主流程里由一组 `once` 分支构成贯穿全年的 peer 弧，基调是同级好兄弟之间「能较真较真、该兜底兜底」的正向搭档情：

- **第一话（monthIndex 1）· 求助埋种**（分支 `peer-zhao-help`）：他卡在因子回测，玩家可抽半天帮一把，好感给赵承宇并埋 `helped_zhao` 旗标（决策 `peer-zhao-help-offer`，framework 借陈星禾）。
- **第三话（monthIndex ≥ 3）· 分歧张力**（分支 `peer-zhao-clash`，取代原单次 once 的盘口插话）：一笔数据上和你意见相左，给出三个取向决策——`peer-clash-stand`（把框架摊开跟他核对）/ `peer-clash-yield`（先顺数据）/ `peer-clash-fence`（约好收盘复盘），各自通过决策的 `setsFlags` 把立场写入 `peer_stand`/`peer_yield`/`peer_fence`，并置 `peer_tension` 防重复触发。
- **第五话（monthIndex ≥ 5）· 再加一次在场**（分支 `peer-zhao-mid`）：夏天补一次量化喊话，再摆研究 vs 数据张力，也暴露他信数据胜过框架的毛病（决策 `peer-zhao-mid-check`）。
- **第七话起（monthIndex ≥ 7）· 按取向分流**（分支 `peer-zhao-resolve-stand/yield/fence` 三选一）：依据第三话写下的立场旗标，只触发对应走向——`stand`→和解（他承认你框架拦住他）、`yield`→认同（他快你稳拼成完整图）、`fence`→中性（真坐到一张桌上先对齐再动手）。每条 once，且只在对应旗标为真时触发，所以「吵架/认同/和解」不会串。
- **第九话（monthIndex ≥ 8）· 延迟后果兑现**（分支 `peer-zhao-payback`）：仅当第一话真帮他（`helped_zhao` 为真）才触发。投委会临时缺人，他主动跟牵头人推荐你顶上，把机会往你这边推——延迟兑现「第一话帮同事」的人情种子。

与 陈星禾 的 `debt-xinghe-seat` 平行：都是「第一话帮同事 → 后期还人情」的延迟种子，引擎统一用「帮同事 + 主要好感对象」埋种，framework 仍借给导师，赵承宇不进图鉴。

### 设计边界（写手必读）
- 赵承宇是 **peer（同级友人同事）**，不进浪漫主线路线（bestRoute 已排除 mentor 筛选），不进研究图鉴教学目录（MENTOR_TEACHINGS 无他，决策 framework 指给导师）。
- 他有一条专属、**纯搭档**的「最佳搭档」结局：条件为 赵承宇关系 ≥ 40（PEER_PARTNER_RELATION）且 teamTrust ≥ 60（PEER_PARTNER_TRUST），与心动/默契浪漫结局**彻底分流**——这是「一个看框架、一个看数据，组合回撤最可控」的并肩战友结局，不是恋爱结局。
- 跨过搭档门槛时，里程碑横幅对 peer 去浪漫化并以 slate 冷色调区分；当前为彩蛋状态，peer 里程碑横幅不弹（常规 UI 不暴露），真图/真线后续由 杂鱼 决定是否唤醒。
- 他的作用是丰富职场生态、提供数据视角的压力测试、暴露自身毛病（追涨、信数据胜过信框架），并演示「能较真也能兜底」的健康好兄弟搭档关系——**不是**「唯一清醒的男的来拯救被女人带偏的你」那套俗套。
- 友谊/搭档线保持正向、健康、符合公序良俗：不蹭浪漫心动线、不写暧昧、不拉满成恋爱式 payoff。
